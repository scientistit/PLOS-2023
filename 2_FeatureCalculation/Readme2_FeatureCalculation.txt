Main Code file of this folder: FeaturesCalculation.
In this code you calculate a set of features. You can easily comment out the features you dont want to calculate - they will not be included in the output set of features.
The example output sets of features are located in \'82../FeatureTables\'92 folder.

The MeanShiftCluster.m file is cited as:
Bart Finkston (2023). Mean Shift Clustering (https://www.mathworks.com/matlabcentral/fileexchange/10161-mean-shift-clustering), MATLAB Central File Exchange. Retrieved January 6, 2023.

ImagesAndMasks folder with all folders with microscopic images and their masks is available on the link which is included in the paper. Download the whole folder and place it in: CodesForThePLOSArticle folder.
